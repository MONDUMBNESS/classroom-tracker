การแจ้งเตือนบนเบราว์เซอร์ (Push Notification)

รับการแจ้งเตือนทันทีผ่านเบราว์เซอร์เมื่อระบบทำงานอยู่และมีกำหนดส่งงานใกล้ถึงกำหนด เพื่อไม่ให้คุณพลาดงานสำคัญ

อนุญาตแจ้งเตือนบนเบราว์เซอร์นี้ ยังไม่ได้ขอสิทธิ์แจ้งเตือน

การแจ้งเตือนผ่านกลุ่ม LINE (LINE Notify)

ตั้งค่าเพื่อคัดลอกสรุปรายการงานค้างสำหรับนำไปโพสต์แจ้งเตือนเพื่อนๆ หรือกลุ่มส่งงานได้อย่างสะดวกและรวดเร็ว

1. สมัครขอรับ Token ได้ฟรีจาก LINE Notify

2. นำ Token ดังกล่าวมาบันทึกในช่องด้านล่างเพื่อประมวลผลคำคัดลอกสเตตัสส่งแจ้งเตือนอัติโนมัติ

จัดการรายวิชาที่ต้องการติดตาม

เชื่อมต่อกับ API ของ Google Console

หากต้องการดึงข้อมูลรายวิชาของคุณจาก Google Classroom จริงๆ คุณสามารถเปิดใช้งานบริการ Google API และนำค่ามาใส่ที่นี่

สิทธิ์ขอบเขตของข้อมูล (Scopes) ที่เปิดใช้งาน:
  • classroom.courses.readonly (ดูรายวิชาที่ลงเรียน)
  • classroom.coursework.me.readonly (ดึงงานทั้งหมดและงานที่ค้างส่งของคุณ)
